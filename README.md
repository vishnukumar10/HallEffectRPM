# HallEffectRPM


Requirements:
-> HallEffect Sensor
-> Arduino Uno or ESP8266 and Arduino software on PC 
-> A few Jumper Cables (Male to Female) 
   or Male to Male with Breadboard

Detailed Explanation at:
